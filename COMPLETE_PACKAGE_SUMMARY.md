## 🆘 TROUBLESHOOTING QUICK LINKS

| Problem | Solution |
|---------|----------|
| Database won't connect | Check PostgreSQL is running, verify DATABASE_URL |
| Port 3000 in use | Kill process: `lsof -ti:3000 \| xargs kill -9` |
| Dependencies won't install | Clear cache: `npm cache clean --force` |
| Docker issues | `docker-compose logs app` to debug |
| Email not sending | Check SMTP credentials in .env.local |

For more help, see QUICK_START_GUIDE.md Troubleshooting section.
